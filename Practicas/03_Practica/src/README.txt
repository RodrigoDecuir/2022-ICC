Para ejecutar los programas:
1. Ir a la carpeta determinada (act01, act02, act03 o act04).
2. Escribir en una terminal 'java ' y presionar tab para autocompletar la linea.
3. Una vez realizado el paso 2, presionar enter para ejecutar el programa.

En el caso de la act04, el programa que se ejecuta es el de cajero, con el mismo procedimiento descrito antes.
