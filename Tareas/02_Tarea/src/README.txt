La carpeta 'src' consta de 6 ejercicios, para ejecutar cada uno es preciso seguir los siguientes pasos:
1.- digitar en el prompt de una terminal 'javac', esto para compilar, seguido del respectivo 'ejercicio.java' que se quiera ejejecutar 
2.- una vez compilado un 'ejercicio.java' se genera un archivo 'ejercicio.class', dicho archivo nos permite ejecutar el programa, por lo cual hay que digitar en la terminal: 'java' seguido de el nombre del ejercicio (sin la terminacion .algo), de hecho si se presiona 'tab' el nombre se autocompleta
3.- una vez realizado el paso 2 hay que repetir 1 y 2 para cada ejercicio 

Para el caso de la calculadora solo es necesario realizar el procedimiento anterior para el archivo "UsoCalculadora.java"
